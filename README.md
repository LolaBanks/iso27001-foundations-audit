# ISO/IEC 27001 Beginner Audit Project

This repository contains **beginner-level ISO/IEC 27001 audit simulations**
created for learning and portfolio purposes.

The project demonstrates foundational understanding of:
- Information Security Management Systems (ISMS)
- ISO/IEC 27001 clauses and Annex A controls
- Risk-based thinking in information security
- Internal audit documentation

⚠️ This is **not a real audit** and does not represent a certified assessment.

---

## Project Objectives

- Understand ISO/IEC 27001 structure and requirements
- Practice basic internal audit documentation
- Learn how risks and controls are mapped
- Build a foundational GRC portfolio

---

## Project Structure

- `iso27001-overview.md` – Overview of ISO/IEC 27001
- `audit-scope-and-context.md` – Defining audit scope
- `risk-assessment-sample.md` – Sample risk assessment
- `statement-of-applicability.md` – Sample SoA
- `internal-audit-checklist.md` – Internal audit checklist
- `nonconformity-examples.md` – Sample findings
- `disclaimer.md` – Legal and ethical disclaimer

---

## Target Audience

- Cybersecurity beginners
- GRC / audit enthusiasts
- Students preparing for ISO 27001 roles
